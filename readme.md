# 🧑‍💼 Human Resource Management System (HRMS)

A full-stack **MERN-based HRMS project** designed to simplify and automate HR operations like attendance, leave management, payroll, and employee administration using separate Employee and Admin dashboards.

---

## 🚀 Features

### 👨‍💻 Employee Panel
- Secure login authentication
- Personal dashboard
- View & update profile
- Mark attendance (Check-In / Check-Out)
- View attendance history
- Apply for leave with reason & dates
- Track leave status (Pending / Approved / Rejected)
- View salary details

### 🛠️ Admin Panel
- Admin authentication
- Manage employee records
- Monitor attendance system-wide
- Approve / reject leave requests
- Manage payroll system
- Full control over HR operations

---

## 🔐 Authentication
- Role-based login system (Employee / Admin)
- JWT-based authentication (if used)
- Protected routes for dashboards

---

## ⚙️ Tech Stack
- Frontend: React.js / HTML / CSS / JavaScript  
- Backend: Node.js + Express.js  
- Database: MongoDB  
- Authentication: JWT / bcrypt  
- Architecture: REST API

## 📌 Workflow

1. Employee logs in
2. Checks in / checks out for attendance
3. Applies for leave request
4. Admin receives request
5. Admin approves/rejects request
6. Employee sees updated status
7. Payroll & attendance data updated automatically

---

## 🎯 Goal

To replace manual HR processes with a **centralized, automated HR system** that improves efficiency, transparency, and accuracy in organizations.
